|                                             GARRISON PUBLIC LIBRARY - A Beacon of Knowledge                                         |
|                                                                                                                                     |
|        The Garrison Public Library (GPL) is a multi-disciplinary library located in Multan, Pakistan. It was generously funde       |
|        by the Government of Punjab and built on 9 acres of prime land donated by the Pakistan Army in Multan Garrison. The          |
|        library is one of the largest public libraries in Pakistan established after independence.                                   |
|                                                                                                                                     |
|        The GPL is a purpose-built modern yet iconic library building with more than 91,544 sqft of covered area. It has three       |
|        floors and unique features such as a children's science museum, day care center, learning commons, rotating deck reading,    |
|        state-of-the-art technology, coffee reading area, community halls and many more.                                             |
|                                                                                                                                     |
|        The library can house half a million print books and accommodate 1000 readers at a time through a variety of seating         |
|        arrangements. The library is manned by a blend of professional and para-professional staff all well qualified and            |
|        experienced workers and leaders dedicated to provide high quality innovative services.                                       |
|                                                                                                                                     |
